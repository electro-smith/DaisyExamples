# ModalVoice Field Example

## Author

Ben Sergentanis

## Description

Demonstrates Modal Voice with 12 tone keyboard and controls over parameters. 
Based on KeyboardTest.

## Controls

| Control | Description |
| --- | --- |
| SW 1 2 |  Octave up/down |
| Keyboard buttons | Chromatic keyboard |
| Knob 1 | Brightness |
| Knob 2 | Structure |
| Knob 3 | Damping |
| Knob 4 | Accent |
