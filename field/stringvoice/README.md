# StringVoice Keyboard

## Author

Ben Sergentanis

## Description

Monophonic string keyboard with controls over string parameters.

## Controls

| Control | Description |
| --- | --- |
| SW 1 2 |  Octave up/down |
| Keyboard buttons | Chromatic keyboard |
| Knob 1 | Brightness | 
| Knob 2 | Structure | 
| Knob 3 | Damping | 
| Knob 4 | Accent |
